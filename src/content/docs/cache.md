Astilba Cache stores the result of expensive server-side work so later calls can reuse it. It is designed for TypeScript applications that need explicit control over invalidation, failures, and who may share a cached value.

> **Development preview:** `@astilba/cache` is not published to npm and has no supported production installation path.

## Explore Cache

Start with the [overview](/docs/cache/overview/), run the [local quickstart](/docs/cache/quickstart/), or inspect the source integrations for [Cloudflare Workers](/docs/cache/cloudflare-workers/) and [React Router](/docs/cache/react-and-server-apps/).

For the precise state of the public source, see [Implementation status](/docs/cache/api-status/).
