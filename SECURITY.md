# Security policy

## Supported version

Security fixes apply to the documentation currently deployed at <https://astilba.com/docs> from the `main` branch. This repository does not publish independently supported release branches.

## Report a vulnerability

Please use [GitHub private vulnerability reporting](https://github.com/astilbahq/docs/security/advisories/new) for vulnerabilities in the documentation site, its Cloudflare Worker, or its deployment pipeline. Include the affected URL or component, the expected impact, and reproduction details when possible.

Do not disclose a suspected vulnerability in a public issue before it has been reviewed.

Until the Cache repository is public, please use this same private reporting form for vulnerabilities in `@astilba/cache` and identify Cache as the affected product.
